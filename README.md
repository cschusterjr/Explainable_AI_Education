
# Explainable AI in Education

**Objective:** Train a tabular classifier and demonstrate explainability (SHAP/LIME-style) and fairness probes.

**Tech Stack:** Python, scikit-learn, SHAP, Streamlit

## Quickstart
```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
python -m src.train
streamlit run app/streamlit_app.py
```
